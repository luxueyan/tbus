土巴士互联网信息服务（北京）有限公司（以下简称土巴士）成立于2015年，注册资本金5000万元。作为国内专业运作“互联网+金融”模式的互联网金融信息服务平台，土巴士凭借强大的互联网技术与全面而先进的交易结构，将移动互联网、云计算、大数据等各项能够应用于金融领域的技术结合传统金融机构经营管理体系，同国内银行金融机构及专业的保理公司、保险公司、资产管理机构、评估机构建立了深入的战略合作关系，使投资人在获得资产稳健增值的同时享受到卓越的互联网金融信息服务所带来的优质体验。 在国内互联网金融市场蓬勃发展的大时代下，土巴士以金融信息服务为支点，深入各个真实的金融场景，为社会提供高效、透明、安全、便捷的金融科技生活服务，着眼未来，打造金融信息服务新引擎。
