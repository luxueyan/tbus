平台介绍

平台概述

自金网系均瑶集团旗下华瑞金融科技有限公司建设和运营的互联网金融资产交易服务平台。秉承着打造健康金融生态圈、为客户提高效益的原则，自金网致力于为客户提供优质、安全、便捷的金融理财信息服务，做一个能保护消费者、有责任感的平台。 自金网与众多金融投资机构、金融交易所、优质实业资产企业等都有广泛的合作，在优质企业、优质资产的保障前提下，自金网遵循“宁缺毋滥”的原则，做到银行级的风险控制。自金网管理团队主要来自各大银行、互联网金融等金融机构，具备丰富的互联网及金融行业企业管理和投融资运作经验。 自金网，是您安全理财的最佳选择。

愿景

打造以客户为中心的健康金融生态圈、为社会带来效益、做责任感的平台。

核心价值

安全：自金网建立了一套成熟的风控体系，遵循“宁缺毋滥”的原则，对每一笔资产、每一个项目都有严格的风控把握。同时，自金网严格按照监管要求，采用资金托管模式，切实保障用户资金安全。
服务：自金网拥有完善的客服体系，竭力为用户提供贴心的服务。自金网以“客户为中心”，制定客户服务标准，为客户提供优质、贴心的服务，努力建造让客户买得放心、用得舒心的平台。
收益：自金网与众多优质投融资机构有广泛的合作，在保障客户资金安全的前提下，尽可能的为客户带来理想的理财收益。
平台公告
安全保障
大数据信用评估

自金网自主研发大数据信用评估体系，数据真实可靠，精准评估融资方信用等级，筛选符合平台安全可靠要求的资产。在大数据信用评估体系的前提下，平台还会对身份证、房产证、工作证明、收入证明、营业执照、信用报告等信息进行真实性的审核。审核方式包括但不限于实地审查、网上审核、尽责调查等。

技术安全保障

自金网采用先进的信息加密技术，对用户的敏感信息操作自动加密传输。同时，自金网拥有完善的安全监测体系，可以及时判断出非正常访问并立即做出应对。自金网从交易、资金、风险、隐私四个方面进行全方位实时监控，对用户资金保驾护航。

法律、风控保障

自金网的每一个项目都经过风控把关，保障项目的低风险性；每一份合同都经过法务的严格审核，切实保护用户的利益。自金网建立了一套成熟的风控体系，遵循“宁缺毋滥”的原则，对每一笔资产、每一个项目都有严格的风控把握。同时，自金网严格按照监管要求，采用资金托管模式，切实保障用户资金安全。
